# Personal-Journal Website üìù

This is Boss Challenge 4 from The Complete Web Development 2020 Bootcamp Course by Angela Yu and version 2 of the blog website created in Boss Challenge 3. It involves creating a full-stack web application using Node.js, Express.js, and MongoDB for the backend. EJS templating and layout functionality are utilized to improve code organization. The project utilizes the MongoDB database to incorporate data storage capabilities. This version also includes a compose button added to the navbar to write posts and a delete button incorporated with every post to delete entries from the database.
